# Next.js App Router + CSS

Scaffolded by [msdkx CLI](https://www.npmjs.com/package/@msdkx/cli).

**Stack:** Next.js 16 · React 19 · TypeScript 6 · CSS Modules

## Get started

```bash
pnpm dev      # start dev server with Turbopack
pnpm build    # production build
pnpm start    # start production server
pnpm lint     # run linter
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project structure

```
src/app/
  layout.tsx        # root layout
  page.tsx          # home page
  page.module.css   # page styles (CSS Modules)
  globals.css       # global reset and base styles
```
