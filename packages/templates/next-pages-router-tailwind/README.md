# Next.js Pages Router + Tailwind CSS

Scaffolded by [msdkx CLI](https://www.npmjs.com/package/@msdkx/cli).

**Stack:** Next.js 16 · React 19 · Tailwind CSS 4 · TypeScript 6

## Get started

```bash
pnpm dev      # start dev server with Turbopack
pnpm build    # production build
pnpm start    # start production server
pnpm lint     # run linter
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project structure

```
src/
  pages/
    _app.tsx        # app wrapper
    _document.tsx   # HTML document
    index.tsx       # home page
  styles/
    globals.css     # global styles + Tailwind import
```

## Tailwind CSS

This template uses Tailwind CSS v4 with the CSS-first configuration.

```css
@import "tailwindcss";
```
